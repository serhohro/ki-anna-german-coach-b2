# 🇩🇪 Анна — German Coach B2

Интерактивный помощник для подготовки к экзамену Goethe/ÖSD B2.  
Словарь, перевод фраз, тесты, голосовой ввод и озвучка.

---

## 🇷🇺 Русский

### О программе

**Анна** — это виртуальный преподаватель немецкого языка уровня B2.  
Программа помогает:
- учить слова с примерами и переводом
- переводить фразы с русского на немецкий
- проверять знания в тестовом режиме
- практиковать устную речь через голосовой ввод

### Функционал

| Режим | Описание |
|-------|----------|
| 💬 Чат | Общение с Анной на немецком |
| 📚 Словарь | 500+ слов B2 с примерами, переводом и кнопкой озвучки |
| 🎯 Тест | Проверка перевода слов, подсчёт правильных ответов |
| 🎤 Голос | Перевод фраз (текст или голос) с озвучкой на немецком |

**Дополнительно:**
- 🔊 Озвучивание слов, примеров и ответов (TTS)
- 🎙️ Распознавание речи через микрофон
- 📊 Прогресс изучения слов

### Как запустить (локально)

1. Установите Python 3.8+
2. Установите зависимости:
   ```bash
   pip install streamlit gtts speechrecognition
   ```
3. Запустите:
   ```bash
   streamlit run anna.py
   ```
4. Откройте в браузере `http://localhost:8501`

**Anna** ist eine virtuelle Deutschlehrerin für das B2-Niveau.  
Die App hilft:
- Vokabeln mit Beispielen zu lernen
- Sätze vom Russischen ins Deutsche zu übersetzen
- Wissen im Testmodus zu prüfen
- gesprochene Sprache über Spracheingabe zu trainieren

### Funktionen

| Modus | Beschreibung |
|-------|--------------|
| 💬 Chat | Auf Deutsch mit Anna sprechen |

**Zusätzlich:**

## 🇬🇧 English

### About the app

**Anna** is a virtual German coach for B2 level.  
The app helps you:
- learn vocabulary with example sentences
- translate phrases from Russian to German
- test your knowledge with quizzes
- practice speaking via voice input

### Features

| Mode | Description |
|------|-------------|
| 💬 Chat | Communicate with Anna in German |
| 📚 Dictionary | 500+ B2 words with examples, translation and pronunciation |
| 🎯 Quiz | Check word translation, track your score |
| 🎤 Voice | Translate phrases (text or voice) with speech output |

**Additional:**
- 🔊 Text-to-speech for words, examples and answers
- 🎙️ Speech recognition (microphone)
- 📊 Learning progress

### Local launch

1. Install Python 3.8+
2. Install dependencies:
   ```bash
   pip install streamlit gtts speechrecognition
   ```
3. Run:
   ```bash
   streamlit run anna.py
   ```
4. Open in browser: `http://localhost:8501`
